# NgxMugenScroll

Inifinit scroll library for Angular

## Development

### Get depends to develop

```bash
npm install
```

### Run library on local demo application

Test library on demo* application.

```bash
# A terminal
npm run build-lib
# Another terminal
npm run start-demo1
```

### Unit test

```bash
npm run test-lib
```

### Doc

```bash
npm run start-docs
```

### Publish

```bash
npm run build-dist
cd dist/ngx-mugen-scroll
npm publish --tag beta
```
